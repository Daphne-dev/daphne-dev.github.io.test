CSS Selector

element에 style 지정을 위한 3가지 기본 선택자

  • tag로 지정
  • id로 지정
  • class로 지정

CSS Selector의 다양한 활용

  • 그룹 선택(여러 개 셀렉터에 같은 style을 적용할 때)
  • 자식 선택 (>) : 자식은 바로 하위엘리먼트를 가리킨다.
  • 아래의 경우 span tag 2만 red 색상이 적용된다.
  • n번째 자식요소 선택(nth-child)
  • 첫번째 단락입니다. 에 red 색상이 적용

nth-child vs nth-of-type

  • nth-child(n) : 태그 상관 없이 n번째를 찾아서 적용(자식 태그)
  • nth-of-type(n) : 같은 태그의 n번째를 찾아서 적용
